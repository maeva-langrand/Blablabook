# BlaBlaBook

BlaBlaBook est une plateforme en ligne dédiée à la découverte et au partage de livres. Son objectif est de favoriser les échanges autour des ouvrages, et de créer une communauté autour de la littérature. BlaBlaBook se veut intuitif et facile d’utilisation.

## À l'attention des développeurs

- N'oubliez pas de créer votre .env à partir du fichier `.env.example`
- `npm i`
- `npm run dev`
- `npm run db:create`
- `npm run db:seed`
